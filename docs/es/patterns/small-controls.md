# Controles pequeños

Los controles pequeños incluyen sliders, step sliders, botones pequeños, toggles tipo checkbox, icon buttons y campos compactos.

## Button

Usa button cuando el control ejecuta un comando.

```text
press -> event -> action
```

## Toggle

Usa toggle cuando el control posee estado persistente on/off.

```text
checked / unchecked -> estado visual distinto
```

## Slider

Usa slider o step slider cuando el jugador elige un número o una opción. Si está dentro de un custom form, documenta el orden de respuesta del servidor.

## Icon Button

Los icon buttons van bien para close, back, next, previous, refresh y limpiar búsqueda.

## Tamaño

Aunque sean pequeños, los controles necesitan un área usable. Si debe tocarse en móvil, no lo hagas visualmente diminuto.
