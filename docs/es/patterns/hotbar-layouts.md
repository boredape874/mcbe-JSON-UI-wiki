# Layouts de hotbar

Modificar la hotbar cambia cómo el jugador ve los slots rápidos y controles relacionados del HUD.

## Formas comunes

- hotbar horizontal compacta
- hotbar vertical lateral
- ilusión circular o radial
- frame personalizado para el slot seleccionado
- cluster de estado cerca de la hotbar

## Regla práctica

La hotbar no debe bloquear el gameplay. Tiene que leerse bien mientras el jugador se mueve o pelea.

## Consejo de layout

Mantén tamaño de slot, gap y textura de selección estables. Evita que el hover cambie el tamaño del slot si eso mueve toda la barra.

## Riesgo

La hotbar toca comportamiento central del HUD. Si el pack apunta a varios dispositivos, prueba touch, controller y teclado/ratón.
