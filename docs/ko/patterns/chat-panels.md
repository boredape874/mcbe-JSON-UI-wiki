# 채팅 패널

채팅 UI 패턴은 서버가 보낸 메시지를 일반 채팅 줄이 아니라 시각 패널로 보여주고 싶을 때 사용합니다.

## 용도

- 숨김 프로토콜 메시지
- 명령 팔레트 힌트
- 알림 패널
- 대화 텍스트
- 서버 공지 스타일링

## 프로토콜 메시지 숨기기

흔한 흐름은 아래와 같습니다.

```text
서버가 marker text 전송
        -> chat UI가 marker 감지
        -> 원본 채팅 줄 숨김
        -> custom panel에 파싱된 내용 표시
```

일반 플레이어가 우연히 입력하기 어려운 marker를 사용하세요.

## 일반 채팅 유지

내 프로토콜 메시지가 아니라면 기존 채팅은 그대로 보여줘야 합니다. 채팅 UI 전체를 무리하게 갈아엎으면 일반 채팅 사용성이 망가집니다.

## 리스크

채팅은 강한 구조화 전송 수단이 아닙니다. 가벼운 표시에는 좋지만 중요한 서버 상태에는 쓰지 않는 편이 안전합니다.
