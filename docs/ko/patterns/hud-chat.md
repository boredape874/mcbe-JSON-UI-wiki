# HUD와 채팅

HUD와 채팅 UI는 실시간 오버레이에 자주 사용됩니다.

## HUD

HUD는 계속 보여야 하는 정보에 적합합니다.

- 체력바
- 마나바
- 퀘스트 표시
- 스코어보드형 오버레이
- 서버 상태 패널

## 채팅

채팅 UI는 특정 메시지를 숨기거나, 서버가 보낸 특수 메시지를 시각 패널로 바꾸는 데 사용할 수 있습니다.

## 타이틀과 액션바

서버가 구조화된 타이틀 또는 액션바 텍스트를 보내고 JSON UI가 일부 값을 파싱해 표시하는 패턴도 자주 씁니다.

서버폼과 다른 점은 서버폼은 플레이어 입력을 서버로 돌려주는 화면이고, HUD와 채팅은 보통 상태 표시나 메시지 표시 중심이라는 점입니다.
