# HUD와 채팅

HUD와 채팅 UI는 실시간 오버레이를 만들 때 자주 사용됩니다.

## HUD

계속 보여야 하는 정보에는 HUD 수정을 사용합니다.

- 체력바
- 마나바
- 퀘스트 추적 표시
- 스코어보드 스타일 오버레이

## 채팅

채팅 UI는 특정 메시지를 보여주거나 숨기는 데 사용할 수 있습니다. 일부 팩은 숨겨진 텍스트 마커를 가벼운 통신 규칙처럼 사용합니다.

## 타이틀과 액션바

많은 서버는 구조화된 title/actionbar 텍스트를 보냅니다. JSON UI는 바인딩으로 그 텍스트의 일부를 파싱해서 커스텀 패널에 표시할 수 있습니다.

## 주의

강한 안정성이 필요한 데이터까지 채팅이나 액션바 프로토콜에 과하게 의존하지 마세요. 복잡한 시스템은 JSON UI 표시와 서버 측 검증을 함께 사용해야 합니다.
