# 레이아웃 기초

Bedrock JSON UI의 레이아웃은 주로 아래 속성으로 결정됩니다.

- `size`
- `anchor_from`
- `anchor_to`
- `offset`
- 부모와 자식 컨트롤 구조

## Size

```json
"size": [120, 40]
```

너비 `120`, 높이 `40`이라는 뜻입니다.

문자열로 쓰는 값도 있습니다.

```json
"size": ["100%", "100%"]
```

부모 컨트롤의 전체 크기를 사용한다는 뜻입니다.

## Anchor

```json
"anchor_from": "center",
"anchor_to": "center"
```

앵커는 "내 컨트롤의 어느 지점을 부모의 어느 지점에 붙일지" 결정합니다.

## Offset

```json
"offset": [0, -20]
```

앵커로 붙인 뒤 위치를 추가로 이동합니다. `x` 양수는 오른쪽, `y` 양수는 아래쪽입니다.

## 부모 크기가 중요함

자식 컨트롤은 부모를 기준으로 배치됩니다. 부모 크기가 `[0, 0]`이면 `"100%"`를 쓰는 자식 컨트롤이 이상하게 동작할 수 있습니다.

## 실전 규칙

입문 단계에서는 중요한 패널마다 안정적인 크기를 주세요.

```json
{
  "type": "panel",
  "size": [300, 180],
  "anchor_from": "center",
  "anchor_to": "center"
}
```

구조가 정상적으로 보이는 것을 확인한 뒤 반응형 배치로 바꾸는 편이 안전합니다.
